# Stock Portfolio Tracker (Python)

A simple Python project that calculates total investment based on user input and predefined stock prices.

---

## Features
- Predefined stock price dictionary
- User can enter stock name and quantity
- Calculates individual investment per stock
- Tracks total investment
- Saves data to a text file

---

## Concepts Used
- dictionaries
- loops
- input/output handling
- arithmetic operations
- file handling

---

## How to Run

```bash
python stock_tracker.py
